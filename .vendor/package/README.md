## 使用

建议使用 [pro-api-sdk](https://gitee.com/jlceda/pro-api-sdk) 进行扩展开发，本定义文件已在 SDK 内预配。

1. 安装类型定义

    ```shell
    npm install --save-dev @jlceda/pro-api-types
    ```

2. 将 `./node_modules/@jlceda/pro-api-types/` 添加到 `tsconfig.include` 内
